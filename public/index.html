What is CoGraphics

CoGraphics is a great app for image processing,
and powered by GCImageSDK which provides developers the leading global mobile graphics technology.

More Information

Powerful and various kinds of filter family CoGraphics has rich and various kinds of filter family. Currently more than 6 filter families and 45 filters can be found in the app, such as, City Family, Blend Family, Distort Family, Blur Family, Frame Family, Postcard Family. Much more and newer filter families/individual filters could be updated online.

Wonderful and colorful themes There are many beautiful themes in CoGraphics, including the default three themes, Deep Space Grey, Clean Fresh Green and Cherry Pink, etc. The powerful online update function could give you much more and beautiful themes.

Convenient filter selection modes
Switch filter by tapping grids in the toolbar panel.
Switch filter by swiping screen much easierly.

Varied and wonderful animation effects
A wonderful animation effect could be played while user taps filter toolbar panel to switch filter;
The animation duration and direction could also be customized in the settings.

Support more than 30 languages
The app provides users a complete i18n and l10n environment.
GCImageSDK is produced by the leading global graphics technology team - Graphics China.

GCImageSDK
GCImageSDK provides iOS apps a powerful, efficient and stable graphics engine. With the engine, any iOS apps could be capable of advanced image processing technology easily, and much more beautiful and attractive. Users might spend much more time on the apps.

GCImageSDK provides app developers several ways to integrate into their apps,
SDK+UI mode: App developers can make their apps to have powerful and efficient image processing capability easily, even just 2~3 line code.
SDK mode: App developers can create and organize their own style UI by themselves, and then using the powerful and complete interfaces provided by the SDK to handle the whole image processing procedure.

GCImageSDK provides many kinds of image processing capabilities, such as
color,
frame,
blend,
distort
and crop, etc.
